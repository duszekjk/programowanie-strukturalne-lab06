# Tasks for students

1. Compile one of the examples from `examples/`.
2. Write a small C program that sends a sequence of commands to `/tmp/ue_character_commands`.
3. Build a path using repeated single-step impulses instead of long movement parameters.
4. Test `FORWARD`, `RIGHT`, `JUMP`, `RESET`, `STOP`, and `PING`.
5. Verify that invalid text is ignored and logged by Unreal.
6. Run the game once with only IPC control and once with keyboard input, then compare the auto/manual counters.
7. Fill in `Config/SubmissionConfig.json` with your index, group number, and optional name fields.
